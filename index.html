Disco-Ball
Disco-Ball is an interative discord bot developed in python for users on discord looking for entertainment and server management. Developed and hosted on AWS by Humza Ahmad

Disco-Ball Features

Play music and youtube vidoe audio directly into a voice channel where you and your friends can listen to for hours!
Control your server through easy to use commands that will ensure rules are followed and enforced, by those who implement them
Create fully function timed polls with a live countdown. Figure out what opinions your server has and see who voted for who!
Keep server entertained through fun minigames, such as Battleships, Rock-Paper-Scissor, etc.
